Book a Meal

An app where a user can create an account, login and book a meal by selecting from the menu available for a given day. If a user is logged in, he/she has the ability to see the day's menu and make an order based on the menu for the day. He/she is able to pay for the meal, see his/her order history and get notifications when the menu for the day has been changed.
